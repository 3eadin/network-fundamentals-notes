# DNS (Domain Name System)

## What is DNS?

DNS translates domain names into IP addresses.

Example:
google.com → 142.250.xxx.xxx

## Why DNS is Important

* Humans use names, not IPs
* Makes Internet easier to use

## How DNS Works

1. User enters a domain name
2. DNS server resolves it to an IP
3. Device connects to the IP

## Example

www.google.com → IP address → website loads

## What I Learned

DNS acts like a phonebook for the Internet, converting names into IP addresses.
