IP Addressing 

What is an IP address

An IP address is a unique indentifier to a device on a network 

Types of IP address 

- IPv4
- IPv6

IPv4 Structure 
- 32 bits
- written as 4 octets (e.g. 192.168.1.1)

IPv6 Structure 
- 128 bits
- Written in hexadecimal (0-9, a-f)
- separated by colons (e.g. 2001:0db8:85a3:0000:0000:8a2e:0370:7334)

Public vs Private IP 
- Private: 192.168.x.x, 10.x.x.x, 172.16.x.x
- Public: routable on the Internet

What I learned 
IP addresses are use to identify devices and allow communication between them. 

Notes 
Understanding IP addressing is essential for subnetting and routing 
IP address is like a home address for a device on a network
