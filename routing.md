# Routing

## What is Routing?

Routing is the process of forwarding packets between different networks.

## Static vs Dynamic Routing

### Static Routing

* Manually configured by the administrator
* Fixed paths
* No automatic updates

**Advantages:**

* Simple
* More secure
* Low resource usage

**Disadvantages:**

* Not scalable
* Requires manual changes

---

### Dynamic Routing

* Uses routing protocols (e.g. OSPF, RIP, EIGRP)
* Automatically updates routes
* Adapts to network changes

**Advantages:**

* Scalable
* Automatic updates
* Efficient path selection

**Disadvantages:**

* More complex
* Uses CPU and bandwidth

---

## Key Difference

| Static Routing | Dynamic Routing       |
| -------------- | --------------------- |
| Manual         | Automatic             |
| No updates     | Updates automatically |
| Small networks | Large networks        |

---

## What I Learned

Static routing is simple but limited, while dynamic routing is more flexible and scalable.
