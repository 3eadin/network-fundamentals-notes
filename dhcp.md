# DHCP (Dynamic Host Configuration Protocol)

## What is DHCP?

DHCP automatically assigns IP addresses and network settings to devices.

## What DHCP Provides

* IP address
* Subnet mask
* Default gateway
* DNS server

## DHCP Process (DORA)

1. Discover → Client looks for DHCP server
2. Offer → Server offers an IP address
3. Request → Client requests the offered IP
4. ACK → Server confirms the assignment

## Advantages

* Automatic configuration
* Reduces manual errors
* Saves time

## What I Learned

DHCP simplifies network configuration by automatically assigning IP addresses.
